# BugTrackr

BugTrackr is a simple React app to add, view, resolve, and delete bug reports. Bugs are stored in localStorage, making it easy to test without backend setup.

## Features
- Add new bug with title and description
- Toggle bug resolved state
- Delete bugs
- Responsive and clean UI

## Setup
1. Clone repo
2. Run `npm install`
3. Run `npm start` to develop locally
4. Run `npm run build` to create production build
5. Deploy to Vercel by linking your repo and following Vercel steps
